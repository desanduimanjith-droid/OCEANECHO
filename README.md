# OCEANECHO
Project Overview
This project serves as a personal milestone in my web development journey, representing a foundational piece of work I originally crafted back in 2024. Using a combination of HTML5 and CSS3, I built this webpage to explore the core principles of front-end design, responsive layouts, and clean code architecture. While the project was conceived and developed a couple of years ago, I have recently migrated it to GitHub to preserve my progress, document my growth as a developer, and share my early design logic with the open-source community.
